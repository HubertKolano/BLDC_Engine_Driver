SET:VOLTAGE %d       sets the output voltage [mV]
SET:RPM %f           controls the setpoint to the desired value
SET:WIFI %s %s       stores the Wi-Fi SSID and password (requires device restart)
SET:DIRECTION        changes the rotation direction

GET:WIFI             Wi-Fi status; if enabled, shows the IP
GET:VOLTAGE          shows the output voltage of the circuit [mV]
GET:RPM              shows the measured RPM of the motor
